# প্লোকিরেজলুশন v2.0
# PlokyResolution System v2.0

> **সমার্থন:** প্লোকিমার্কেটের রিজলুশন সিস্টেম। বাংলাদেশের যোগসর্থের জন্য নির্মিত।
> **Standard:** Industry-standard prediction market resolution system for Bangladesh.

## সমন্ভন্ত সমস্যা এবং সমাধান

### কি কি সমস্যা অছিল? কি কি সমাধান অছিল?

| সমস্যা | কিভাবে সমাধান করা হয়েছে |
|---------|--------------------------------|
| একজন রিজলভার compromise হলে ভুল verdict | Multi-sig + Reputation + Stake |
| সল নির্ণয় হস্তাখেপের উপর নির্ভর | AI এভিডেন্স অ্যানালাইজার |
| কোনো অর্থনৈতিক প্রণোদনা নেই | Bond + Slashing + Rewards |
| নির্দিষ্ট সময় কিছু নেই | Escalation Layer (AI → Community → Expert → UMA) |
| রিজলভারের নির্দিষ্টে কোনো জবাবধা নেই | Reputation Scoreboard + Stake |
| কোনো সার্ভার-সাইড ইন্ডিক্সিঙ নেই | The Graph Subgraph + Redis Cache |
| নির্দিষ্ট সময় সিদ্ধান্ত নির্ধারণ হলে | Timelock + Auto-freeze + Admin |

## সম্পর্ক প্রয়োজনীয় (বাংলাদেশ বিশেষ)

### গ্যাস সমস্যা কমানোর জন্য
- **Polygon PoS** এর gas fee অতি কম ($0.001-$0.01)
- প্রতি ট্রানজাকশ঩ কম হলে বাংলাদেশী যোগসর্থের জন্য সহজ

### ভাষা সমর্থন
- সমস্ত UI বাংলায় এবং ইংরেজিতে দুইটিয়া লেন্গুয়েজ সাপোর্ট
- বাংলাদেশী খবর সার্বসামুলক তথ্য বিবেচনা

### মোবাইল-ফার্স্ট
- 80% বাংলাদেশী যোগসর্থ মোবাইল দিয়ে ইন্টারনেট ব্যবহার করে
- মোবাইল-অনুকূল UI ডিজাইন প্রয়োজন

### রেগুলেটরি সমস্যা
- বাংলাদেশ ব্যাঙ্ক এর সাথে কাজ করা
- BSEC গাইডলাইনের সাথে সামঞ্জস্য রাখা
