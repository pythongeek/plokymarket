# Plokymarket Monorepo Template

This repository contains a skeleton monorepo structure utilizing Next.js for the web workspace.

## Project Structure

- `apps/web`: Next.js web application frontend skeleton.
- `LICENSE`: Project license.

## Getting Started

To install dependencies:

```bash
npm install
```

To run the web app development server:

```bash
npm run dev --workspace=apps/web
```
